Views.File

extends Backbone.View

Internal: View that renders a File node and controls its behavior (class: 'file', tag: 'li').

Instance Methods

initialize(options)

Internal: Initialize a new file node. Adds associated model cid to the view element. Binds change handlers to the change:name and change:extension events. These take care of resorting the file nodes.

  • options - Passes in settings object, which is used to control whether or not file extensions are shown.

render()

Internal: Sets the text of the file node according to the underlying model name. If the 'showExtensions' setting is set, renders the full file name with extension, otherwise renders just the file name attribute.
