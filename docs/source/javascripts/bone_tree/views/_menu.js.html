Classes

Views.Menu

extends Backbone.View

Internal: View that controls the context menu (class: 'filetree_context_menu').

Events

  • contextMenu - Prevents the standard browser context menu from appearing when right clicking within the file tree context menu.

  • click .rename - Prompts the user to rename a file.

  • click .delete - Deletes a node from the file tree.

Instance Methods

initialize(options)

Internal: Initialize a new menu widget.

  • options - An Object. Internally used to pass the settings configuration into the menu. This controls whether or not the user is prompted to confirm deleting a file.

contextMenu(e)

Internal: Kill the default browser behavior for the contextmenu event.

delete(e)

Internal: Deletes a node from the file tree. If the confirmDeletes setting is set, prompts the user for delete confirmation.

rename(e)

Internal: Prompts the user to rename a File or Directory.

render()

Internal: Renders the

    that contains the context menu choices 'Rename' and 'Delete'.

    Returns this, the menu view.

template()

Internal: html template for the context menu.
